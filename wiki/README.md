## 🔍 Understanding the Application

### How is it built?
- **Vanilla JavaScript** (no frameworks)
- **HTML5 & CSS3** (modern standards)
- **SVG Graphics** (scalable characters)
- **Local Storage** (offline persistence)
- **No build process** (direct browser execution)

### Who can contribute?
Anyone! We welcome:
- 🐛 Bug reports
- ✨ Feature ideas
- 📝 Documentation improvements
- 🎨 Design suggestions
- 💻 Code contributions

### What's the philosophy?
- **Simple**: No unnecessary complexity
- **Educational**: Learn from the code
- **Accessible**: Works on any device
- **Private**: Your data stays on your device
- **Offline**: Works without internet

## ❓ FAQ

### Q: Do I need to install anything?
**A**: No! Just open the HTML file directly in your browser. No npm, no build process, no server required.

### Q: Does it work offline?
**A**: Yes! The entire app runs offline. All data is stored on your device.

### Q: Is my data private?
**A**: Completely! All data stays on your device. Nothing is sent to any server.

### Q: Which browsers are supported?
**A**: All modern browsers (Chrome, Firefox, Safari, Edge). Works on desktop, tablet, and mobile.

### Q: Can I contribute?
**A**: Absolutely! See the **[Development Guide](./Development.md)** for contributing guidelines.

### Q: What devices does it work on?
**A**: Any device with a web browser (Windows, Mac, Linux, iOS, Android).

### Q: How much data does it use?
**A**: ~50-100 KB for saved progress. Very lightweight!

### Q: Can I backup my progress?
**A**: Progress is saved in your browser. You can export it (feature coming soon).
